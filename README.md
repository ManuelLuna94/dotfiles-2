# dotfiles
These are my dotfiles for my current system.

## Requirements
**Desktop Environment** - i3-gaps
**Browser** - Chrome
**Title Bar** - Polybar
**Lock** - i3lock
**Terminal** - Konsole

## For lockscreen
Paste the lock.sh file in /bin/ folder.

## For wallpaper
Put your wallpaper with name '.wallpaper.png' in home directory.
